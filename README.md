### Introdução

##### Nesse artigo será aprensentado os modelos, vantagens e desvantagens em se utilizar a arquitetura de Von Neumann ou a Arquitetura de Harvard. Cada uma dessas arquiteturas trabalha com diferentes modelos de organização visando melhorar a eficiência das máquinas dependendo da sua finalidade. Do ponto de vista de padronização pode-se afirmar que a maioria dos computadores atuais baseam-se na arquitetura de Von Neumann que se basea no principio de que é possível armazenar um programa na mesma memória de dados.
